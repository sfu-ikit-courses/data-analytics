1. Что такое статистика? Примеры использования статистики в реальной жизни.¶

Статистика — это набор математических процедур, которые необходимы для организации, обобщения и интерпретации информации.

Примеры использования:

  • Анализ рыночных данных в бизнесе: прогнозирование спроса, анализ потребительского поведения, оптимизация ценовой политики.

  • Оценка демографических показателей: анализ численности населения, структуры возрастных групп, миграции.

  • Медицинские исследования: изучение эффективности лекарств, анализ распространения заболеваний, статистика смертности и заболеваемости.

2. Типы признаков: количественные и качественные.¶

  1. Количественные данные выражаются числами, которые имеют математическое значение (рост, вес, зарплата). Такие данные можно сравнивать друг с другом.

    Существует два подтипа количественных данных:

    • Интервальные данные. Нулевое значение выбирается произвольно для удобства. Например, температура по Цельсию и по Фаренгейту.
    • Данные, измеренные по шкале отношений. Нулевое значение определяется однозначно. Например, количество товара на складе.

    Также количественные данные можно разделить на две других подгруппы:

    • Дискретные
    • Непрерывные

  2. Качественные данные отражают принадлежность к категориям.

    Существует два подтипа качественных данных:

    • Порядковые. Их можно упорядочить. Например, степень ожога.
    • Номинальные. Нельзя упорядочить. Например, породы кошек.

3. Количественные данные: интервальные данные и данные, измеренные по шкале отношений.¶

Количественные данные выражаются числами, которые имеют математическое значение (рост, вес, зарплата). Такие данные можно сравнивать друг с другом.

  1. Интервальные данные. Нулевое значение выбирается произвольно для удобства

    • pH-уровень раствора (0 pH не означает отсутствия кислотности, это просто граница шкалы);
    • Год рождения (разница между 1990 и 2000 годом имеет смысл, но "нулевой" год в календаре условный).

  2. Данные, измеренные по шкале отношений. Нулевое значение определяется однозначно.

    • Расход топлива автомобиля в литрах (0 литров — значит топлива нет);
    • Время в секундах (0 секунд — это полное отсутствие продолжительности события).

4. Количественные данные: непрерывные и дискретные.¶

  1. Дискретные. Принимают только конкретные (целые) значения, не могут быть дробными.

    • Количество детей в семье;
    • Число автомобилей на парковке;
    • Количество экзаменов.

  2. Непрерывные. Могут принимать любые значения в пределах диапазона, в том числе дробные.

    • Время бега на 100 метров;
    • Средний балл в классе (по 5-балльной шкале);
    • Температура на улице.

5. Качественные данные: порядковые и номинальные.¶

  1. Порядковые. Категории можно упорядочить.

    • Размер одежды (S, M, L, XL);
    • Уровень прожарки мяса (Extra Rare, Rare, Medium Rare, Medium Well);
    • Воинские звания (рядовой, сержант, лейтенант, капитан, майор).

  2. Номинальные. Категории без четкого порядка.

    • Цвет глаз (синий, карий, зеленый);
    • Гражданство (Росии, Германии, Франции, США);
    • Виды джинсов (Клеш, скинни, мом, широкие).

6. Что такое генеральная совокупность?¶

Генеральная совокупность — это совокупность всех объектов, которые представляют интерес в конкретном исследовании.

Примеры:

  • Все студенты в городе;
  • Все пациенты с диабетом в стране.

7. Что такое выборка?¶

Выборка — это группа объектов, отобранных из генеральной совокупности для исследования.

Примеры:

  • Часть студентов, которых отобрали для измерения роста;
  • 200 пациентов, принявших участие в исследовании.

8. Репрезентативность выборки. Смещение.¶

Репрезентативность — соответствие характеристик выборки характеристикам генеральной совокупности.

Смещение выборки — явление, при котором статистические характеристики выборки сильно отличаются (смещены) относительно характеристик генеральной совокупности.

10. Что такое случайная величина (СВ)?¶

Случайная величина (СВ) — это величина, значение которой зависит от случайных обстоятельств, то есть она может принимать различные значения с определённой вероятностью. Случайная величина является числовым отображением исходов случайного эксперимента.

В зависимости от множества возможных значений, случайные величины делятся на дискретные и непрерывные.

  • Дискретная случайная величина может принимать только конечное или счётное множество отдельных значений. Например, число очков, выпавших при подбрасывании игральной кости. Возможные значения: все числа от 1 до 6 с одинаковой вероятностью.

  • Непрерывная случайная величина может принимать любые значения из некоторого интервала, и их количество — бесконечно. Например, точность, с которой решается оптимизационная задача.

11. Что такое закон распределения СВ?¶

Закон распределения случайной величины — это правило, которое описывает, как распределяются вероятности на возможные значения случайной величины. Он показывает, с какой вероятностью случайная величина примет каждое из возможных значений (для дискретной СВ) или попадёт в какой-либо интервал значений (для непрерывной СВ).

12. Числовые характеристики дискретных СВ.¶

  1. **Математическое ожидание дискретной случайной величины *X*** — это взвешенная сумма всех возможных значений величины, умноженных на их соответствующие вероятности. Оно отражает среднее значение случайной величины, если бы эксперимент повторялся бесконечно много раз;

  2. Дисперсия измеряет, насколько значения случайной величины отклоняются от её математического ожидания. Это среднее значение квадрата отклонений от среднего значения.

  3. Стандартное отклонение - это корень из дисперсии. Оно также отражает, насколько широко распределены значения случайной величины относительно её среднего значения.

13. Функция распределения СВ, свойства.¶

Функция распределения случайной величины (СВ) — это функция, которая показывает вероятность того, что случайная величина X примет значение, меньшее или равное некоторому заданному числу x. Она описывает весь закон распределения случайной величины.

Свойства:

  • Невозрастающая функция;
  • Функция распределения всегда ограничена снизу и сверху;
  • Для дискретной случайной величины функция распределения является ступенчатой функцией;
  • Для непрерывной случайной величины функция распределения является гладкой и непрерывной, и её производная в точке x (если она существует) является плотностью вероятности.

14. Функция плотности вероятности непрерывной СВ, свойства.¶

Функция плотности вероятности (ПВ) — это функция, которая описывает вероятность того, что непрерывная случайная величина X примет значение, находящееся в определённом интервале. В отличие от дискретных случайных величин, для которых вероятность того, что случайная величина примет конкретное значение, равна нулю, для непрерывных СВ мы говорим о вероятности попадания в интервал.

Свойства:

  • Невозрастающая функция для интервалов вероятности;
  • Интеграл функции плотности вероятности на любом интервале [a,b] даёт вероятность того, что случайная величина X примет значение x в этом интервале;
  • Площадь под графиком функции плотности на всем множестве значений должна быть равна 1.

from scipy import stats


def calculate_statistics(sample, name):
    print(f"Описательные характеристики для {name}:")
    print(f"Минимум: {np.min(sample)}")
    print(f"Максимум: {np.max(sample)}")
    print(f"Математическое ожидание: {np.mean(sample)}")
    print(f"Дисперсия: {np.var(sample)}")
    print(f"Стандартное отклонение: {np.std(sample)}")
    print(f"Квантиль 0.25: {np.quantile(sample, 0.25)}")
    print(f"Квантиль 0.75: {np.quantile(sample, 0.75)}")
    print(f"Размах: {np.ptp(sample)}")
    print(f"Медиана: {np.median(sample)}")
    print(f"Коэффициент асимметрии: {stats.skew(sample)}")
    print(f"Коэффициент эксцесса: {stats.kurtosis(sample)}")


for name, sample in samples.items():
    calculate_statistics(sample, name)
    print("\n")
